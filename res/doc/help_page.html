Справочное руководство по программе

Для захвата поля судоку и его решения необходимо выполнить несколько шагов:

1. Выбор источника изображения.

1.1 Изображение с веб-камеры.
 По умолчанию, при запуске приложения выбран режим захвата изображения с веб камеры. Если веб камера не подключена то в окне приложения ничего отображаться не будет. Подключите веб-камеру, к компьютеру, перезапустите приложение. Если в окне приложения не появилось изображение, попробуйти выбрать другую камеру в настройках приложения (Sudoker->Video capture).



1.2 Изображение из файла.
Открыть изображения из файла можно выбрав в меню Sudoker пункт Image from file (или используя комбинацию клавиш Ctrl + O ).


2. Выбор режима работы.

2.1 Режим распознования.
В этом режиме происходит только захват поля и распознование цифр. Включение этого режима происходит при отключегом режиме решения(Solve).

2.2 Режим решения.
Этот режим можно включить кнопкой Solve на панели инструментов или в меню Mode. Он также включен по умолчанию.


3. Захват поля.
3.1 Для успешного захвата игрового поля должны выполняться несколько условий:

1) Изображение поля должно быть повернуто в плоскости изображения не более чем на 45 градусов. Перевернутое изображение будет распознано не верно.

2) Сторона квадрата игрового поля должна быть больше половины меньшей стороны фокуса камеры.

3) Изображение не должно быть сильно зашумленно.

4) Граница поля не должна быть нарушена инородными объектами.

3.2 Статичное изображение.

Если было выбрано изображение из файла и выполняются все вышеперечисленные требования то поле будет успешно захвачено, распознано и решено.
Если какие либо требования были нарушены поле может быть не захвачено или неправильно распознано.

3.3 Изображение с камеры.

Для захвата судоку с веб-камеры необходимо поместить изображение с полем перед камерой.

Если контур поля отображается красным - значит поле не удается захватить из-за движения в кадре, нужно добиться того чтобы поле изображение было максимальное неподвижно.

Если это не удается сделать можно попробывать уменьшить чуствительность к движению или полностью отключить проверку на движение в настройках (Sudoker->Preferences).

3.4 Продолжение захвата.

В случае успешного захвата игрового поля судоку, приложение переходит в режим просмотра, при этом изображение становится неподвижным. Продолжить работу можно нажав клавишу Esc или кнопку Continue на панели инструментов.
